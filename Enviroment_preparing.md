1. На всех тестовых устройствах запустить скрипт setup_agent, чтобы подготовить их к тестированию
2. Для параллельного тестирования Wi-Fi На роутере iperf должен работать на портах 5201-5221
3. При необходимости, изменить учётные данные в config.py для авторизации в роутер
4. Чтобы удобнее было отслеживать результаты, в настройках системы можно переименовать устройства в user-friendly имена
    Например: DESKTOP-NGM3T1F -> 14X-1, DESKTOP-IDPU7D6 -> 14X-2 и тд
